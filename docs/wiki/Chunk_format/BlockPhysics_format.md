# BlockPhysics format

## Binary format

<div markdown="1" id="table">

| Name | Size (in bytes) | Description |
|------|-----------------|-------------|
| HasData | 1 (boolean) | `true` if support data exists. |
| SupportData | 16384 | 16384 bytes recording the taxicab distance to the nearest support block.<br>4 bits per block (high-nibble-first packed). |

</div>

### Support Logic

* Support distance is measured in [taxicab distance](https://en.wikipedia.org/wiki/Taxicab_geometry).
* A support block:
    * Is any block that doesn't have "physics," i.e., breaking from lack of support,
    * Has a support distance of 0, or
    * Is a decoration block (value of 15).
* Blocks break when the support distance reaches or exceeds 5.
