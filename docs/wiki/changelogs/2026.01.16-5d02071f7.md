# Hytale 2026.01.16-5d02071f7

**2026.01.16-5d02071f7** is a pre-release update released on January 16th, 2026,[\[1\]](#1) which adds some new content and improves performance.

## Known Issues

* lighting / shadow acting up, you can ignore that for now.

## Combat & Balance

* More reliable melee hits
* Burn/Lava tuning + water clears burns
* Fire creatures now fire-immune
* NPC/combat fixes + exploit fixes

## Crafting & Items

* Backpacks are easier + cheaper to craft
* Benches pull from a larger chest radius (2x)
* Faster hide to leather processing (-10s)
* Food/farming balance tweaks
* Dropped-item physics fixes
* You can now place torches while dual-wielding (except sword, it prioritize block)

## Audio & VFX

* New/updated mining + ore sounds
* Improved particles (ore breaking, eating, memories, potions)
* Small visual/positioning fixes

## UI / Performance / QoL

* FOV safeguards + high-FOV fixes
* Faster chunk loading at high view distance
* Pre-release/version warnings
* Better lag handling + small keybind/UI updates

## World & NPCs

* Improved zone 4 atmosphere/spawns (sewers/volcanic areas)
* Villages/outposts/dungeons repaired
* Environmental damage rework
* Earlier sleep time (-1h30m)
* Loot/animation fixes + minor interaction tweaks
* Added dinosaurs in Zone 4

## Customization & Misc

* New face/hair options (unavailable until it goes to release build) + cosmetic fixes
* Cute secret new NPC added for creatives and modders

## Commands / Server / Stability

* Teleport/give/whitelist/ban improvements
* Networking/auth updatesrash + stability fixes
