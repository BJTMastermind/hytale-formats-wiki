# Hytale 2026.01.15-c04fdfe10

**2026.01.15-c04fdfe10** is a hotfix update released on January 15th, 2026,[\[1\]](#1) which addresses more issues reported since launch.

## Fixes

### Connectivity

* General server stability and network improvements
* The game will now switch to offline mode if authentication fails, instead of getting stuck
* Fixed "invalid token" errors that could occur when your system clock was slightly out of sync

### Stability

* Improved client performance by optimizing background task scheduling
* Fixed an issue where dropped items could get stuck inside blocks
* Fixed a server crash that could occur in certain situations
