# Hytale 2026.01.27-734d39026

**2026.01.27-734d39026** is a hotfix update released on January 27th, 2026,[\[1\]](#1) which fixes issues relating to performance and gameplay.

## Fixes

### Performance

* Fixed shadow display issues
* Reduced stuttering and input lag on NVIDIA graphics cards

### Stability

* Fixed several crashes, including ones related to weather, armor equipping, and disconnecting
* Improved login reliability and server stability

### Gameplay

* Fixed compass not pointing to the correct location
* Improved block placing in creative
* Improved teleporter logic to prevent infinitely teleporting
* Fixed crossbow misfires and other interaction
